Projects

  • Person Retrieval in Video Surveillance

    Led the project focused on creating an end-to-end system for person retrieval in video surveillance. Developed a novel person attribute recognition model, and successfully deployed it on Nvidia Jetson Orion hardware.

  • Sleep Stage Classification Using Machine Learning Algorithms

    Designed and developed a machine learning model for sleep stage classification, using various algorithms and Wavelet decomposition of EEG Signal for feature extraction. Achieved an outstanding accuracy rate of 91%.

Research Experience

Project: Person Retrieval in Video Surveillance

Senior Research Fellow (2024–Present) & Junior Research Fellow (2022–2024) at Ahmedabad University, Ahmedabad, India.

Person retrieval in video surveillance is a task of locating a person with the given description in the video frame.

  • Designed and implemented novel person retrieval with novel person attribute recognition architectures such as CorDeepMAR and DensePAR, achieving real-time performance on Nvidia Jetson Orin AGX.
  • Optimized machine learning models for real-time deployment in resource-constrained environments using model compression techniques like quantization and pruning.
  • Developed a Transformer based person retrieval system for natural language queries.
  • Built an annotation tool using the Ultralytics library: AnnoTool.
  • Assisted Prof. Raval in conducting hands-on sessions at various institutions and mentored students.

Masters Dissertation: Sleep Stage Classification Based on Machine Learning Algorithms with Wavelet Decomposition Techniques (2021-2022)

  • Developed a framework for sleep stage classification using machine learning algorithms such as Random Forest and Gradient Boosting Classifiers.
  • Employed wavelet decomposition techniques (Coiflet, Daubechies, and Biorthogonal) for feature extraction from EEG signals.
  • Achieved a classification accuracy of 91% using Hjorth parameters for feature extraction.
  • Code available at: Github: SSC.

B.Tech Dissertation: Energy Management of Hybrid Wind Battery System (2018-2019)

  • Developed statistical and machine learning regression models for wind speed forecasting.
  • Developed a scheduling algorithm for charging and discharging of the battery system based on wind speed forecasts.
  • Reduced the penalty over the power deficit by 40% for three wind farm sites.

Projects

  • Person Retrieval in Video Surveillance

    • Led the project focused on creating an end-to-end system for person retrieval in video surveillance.
    • Developed a novel person attribute recognition model, establishing correlations between attributes.
    • Implemented model compression techniques, reducing its size by 10 times.
    • Successfully deployed the model on Nvidia Jetson Orion hardware for real-time video surveillance applications.

  • Sleep Stage Classification Using Machine Learning Algorithms

    • Designed and developed a machine learning model for sleep stage classification, employing Gradient Boosting, Decision Trees, and Bootstrap Aggregating algorithms.
    • Utilized Wavelet decomposition of EEG Signal to extract features, integrating time domain features for improved accuracy.
    • Tackled imbalanced labels in the dataset through extensive experimentation with under-sampling and over-sampling techniques.
    • Achieved an outstanding accuracy rate of 91% through the application of wavelet decomposition for feature extraction.
